1)Write a Blog on the Difference between Document and window object?
      	Document Object represents web page that is loaded in the browser while Window Object represents a browser window or frame that displays the content of the web page.
      	Document Object is loaded inside the window while Window Object is the very first object loaded in the browser.Document Object is the object of
        window property while Window object is the object of the browser.
      	Document Object is a part of BOM (Browser Object Model) and DOM (Document Object Model) while Window object is a part of BOM (Browser Object Model).
      	Syntax for Document Object is document.propertyname;
      	Syntax for Window Object is window.propertyname;
